## Run Locally

Clone the project

```bash
  git clone https://github.com/TetianaBukhenko/test-task-1.git
```

Go to the project directory

```bash
  cd test-task-1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
